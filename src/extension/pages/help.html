üí° Tips & Best Practices

Recording Tips

  • Use data-testid attributes for the most reliable selectors
  • Record at normal speed - don't rush through actions
  • Test your recordings on different page states
  • Use pause/resume to organize your recording sessions

Generated Script Tips

  • Review and customize generated scripts before use
  • Add additional assertions based on your testing needs
  • Use Page Object Models for maintainable tests
  • Consider parameterizing test data for reusability

üêõ Troubleshooting

Common Issues

  • Recording doesn't start: Refresh the page and try again
  • Actions not captured: Ensure the page is fully loaded
  • Script generation fails: Check your AI API key configuration
  • Elements not found: Enable self-healing for better reliability

Browser Compatibility

FlowScribe works best with:

  • Chrome 88+ (recommended)
  • Chromium-based browsers
  • Latest browser extensions enabled
